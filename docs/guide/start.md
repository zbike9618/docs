# 🔰 スターターガイド

CountryWarsサーバーへようこそ！このページでは、サーバーに参加してから国を作って発展させるまでの基本的な手順を解説します。

## 1. サーバーへの参加

サーバーに入ったら、まずは安全な場所を見つけましょう。初期スポーン地点周辺にはチュートリアルやお店（Shop）があります。
スマホアイテムはShopから購入可能です。

## 2. 建国（CountryWars）の基本

このサーバーの醍醐味である「建国」のアドオン機能です。

### 国の作り方
1. ゲーム内でチャットを開きます。
2. `/cw create [国の名前]` と打ち込みます。
   *(例: `/cw create Japan`)*
3. 成功すると、あなたがその国の「国王」となります！

### 領土の保護
国を作ったら、自分の領土（チャンク）を保護しましょう。
- 保護したい場所に立ち、コマンド `/cw claim` を実行します。
- 保護された領土内では、他国民によるブロック破壊やチェストの開閉が制限されます。（チェスト保護機能つき）

## 3. 仲間を集める

一人で国を発展させるのは大変です。他のプレイヤーを国に招待しましょう。
- 招待コマンド: `/cw invite [プレイヤー名]`
- 招待された側は `/cw join [国の名前]` で加入できます。

## 4. ルールや禁止アイテムについて

当サーバーでは、公平性を保つためのルールがあります。
インベントリに入った瞬間に自動で没収される「禁止アイテム」の一例は以下の通りです。
（特殊なアドオンアイテム等も含まれます）

| アイテム名 | 制限の理由 |
| --- | --- |
| エンチャントされた金のリンゴ | バランス崩壊を防ぐため |
| 不死のトーテム | PVPでの長期戦化を防ぐため |
| 盾 | 戦闘バランス調整のため |

> **💡 TIP:** コマンドの全一覧については、サイドバーの「コマンド辞書」ページもご確認ください！
