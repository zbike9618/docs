## 2. 建国（CountryWars）の基本

このサーバーの醍醐味である「建国」のアドオン機能です。

### 国の作り方
1. ゲーム内でチャットを開きます。
2. `/mc` と打ち込みます。
3. 国名と国家方針を選択します。
4. 送信するとあなたが国王の国が建国されます。

### 領土の保護
国を作ったら、自分の領土（チャンク）を買いましょう。
- 購入したいチャンクで、コマンド `/buyc` を実行します。
- 自国の領土内では、他国民によるブロック破壊やチェストの開閉が制限されます。(権限設定から変更可能)

## 3. 仲間を集める

一人で国を発展させるのは大変です。他のプレイヤーを国に招待しましょう。
- 国設定フォーム`/sc`を開き、国民タブから招待したいプレイヤーを招待します。
- 招待された側は`/mb`から参加できます。

## 4. 職業について

当サーバーでは、プレイヤーの活動を支援するための「職業」システムがあります。
職業に就くと、特定の行動（ブロック破壊、モンスター討伐など）に応じてお金が稼げるようになります。

### 職業の確認と就業
- 職業に関するメニューは、コマンド `/jobs` で開くことができます。
- メニュー内の職業を選択すると就業できます。

### 職業の制限
- 一度に就くことができる職業は **最大4つ** までです。
- 職業を変更したい場合は、現在就いている職業を辞めてから、新しい職業に就く必要があります。

### 職業の例
現在、以下の職業が用意されています。
- **鉱夫 (Miner):** 石や鉱石を採掘することで報酬が得られます。
- **狩人 (Hunter):** モンスターを討伐することで報酬が得られます。
- **林業 (Lumberjack):** 木を伐採することで報酬が得られます。
- **ネザー採掘 (Netherdigger):** ネザーのブロックを採掘することで報酬が得られます。
- **建築家 (Builder):** ブロックを設置・破壊することで報酬が得られます。
