# 🔰 スターターガイド

建国鯖へようこそ！このページでは、サーバーに参加してから国を作って発展させるまでの基本的な手順を解説します。

## 1. サーバーへの参加

サーバーに入ったら、まずは安全な場所を見つけましょう。ロビーのボタンからランダムTPが可能です。
スマホアイテムはShopから購入可能です。

## 2. 建国（CountryWars）の基本

このサーバーの醍醐味である「建国」のアドオン機能です。

### 国の作り方
1. ゲーム内でチャットを開きます。
2. `/mc` と打ち込みます。
3. 国名と国家方針を選択します。
4. 送信するとあなたが国王の国が建国されます。

### 領土の保護
国を作ったら、自分の領土（チャンク）を買いましょう。
- 購入したいチャンクで、コマンド `/buyc` を実行します。
- 自国の領土内では、他国民によるブロック破壊やチェストの開閉が制限されます。(権限設定から変更可能)

## 3. 仲間を集める

一人で国を発展させるのは大変です。他のプレイヤーを国に招待しましょう。
- 国設定フォーム`/sc`を開き、国民タブから招待したいプレイヤーを招待します。
- 招待された側は`/mb`から参加できます。

## 4. ルールや禁止アイテムについて

当サーバーでは、公平性を保つためのルールがあります。
インベントリに入った瞬間に自動で没収される「禁止アイテム」の一例は以下の通りです。
（特殊なアドオンアイテム等も含まれます）

| アイテム名 | 制限の理由 |
| --- | --- |

| 不死のトーテム | PVPでの長期戦化を防ぐため |
