# コマンド一覧

CountryWars内で使用できるコマンドの一覧です。
ゲームの進行に役立つ様々なコマンドが用意されています。

## 基本コマンド

| コマンド | タイトル | 説明 |
|---|---|---|
| `/money` | お金の確認 | 現在の所持金を確認します。 |
| `/bank` | 銀行 | クラウドへの預け入れ、引き出し、両替を行います。 |
| `/jobs` | 職業 | 職業に就くことで、特定の行動でお金を稼ぐことができます。 |
| `cw:pay <金額> <プレイヤー名>` | お金を送る | 指定したプレイヤーにお金を渡します。 |

## 国・領土・戦争関連コマンド

国を作ったり、領土を管理したりするためのコマンドです。

| コマンド | タイトル | 説明 | 簡略化 |
|---|---|---|---|
| `/makecountry` | 国の建国 | 新しく自分の国を建国します。 | `/mc` |
| `/settingscountry` | 国籍・設定 | 所属している国の設定変更や管理を行います。 | `/sc` |
| `/countrylist` | 国リスト | 現在存在する国の一覧を表示します。 | `/lc` |
| `/buychunk` | 土地の購入 | 現在立っている場所のチャンクを自分の領土として購入します。 | `/buyc` |
| `/sellchunk` | 土地の売却 | 所有しているチャンクを売却し、領土を手放します。 | `/sellc` |
| `/checkchunk` | 土地情報の確認 | 現在地の所有者や保護情報を確認します。 | `/cc` |
| `/war` | 戦争 | 他国への宣戦布告や、戦争に関する情報の確認を行います。 | - |
| `cw:invade` | 侵略 | 他国の領土に侵略を開始します。 | - |

## テレポート・ショップ・便利機能

移動やアイテムの売買、メッセージの確認などができる便利なコマンドです。

| コマンド | タイトル | 説明 | 簡略化 |
|---|---|---|---|
| `/sethome` | ホーム設定 | ホームの管理（地点の登録など）ができます。 | - |
| `/home` | ホームへ戻る | 設定したホーム地点へテレポートします。 | - |
| `cw:lobby` | ロビーへ戻る | ロビー（初期スポーン地点等）へテレポートします。 | - |
| `/tpaccept` | テレポート承認 | 他プレイヤーからのテレポートリクエストを承認します。 | `/tpa` |
| `/messagebox` | メールボックス | オフライン中などに届いたメッセージを確認します。 | `/mb` |
| `/playermarket` | プレイヤーマーケット | プレイヤー同士でアイテムを売買できる市場を開きます。 | `/pm` |
| `cw:shop` | ショップ | ショップ画面を開き、アイテムを購入できます。 | - |
| `cw:stock` | 株 | 株関連の操作（株価の確認や売買）を行います。 | - |
| `cw:selfkill` | 自殺 | 自分自身をキル（リスポーン）します。 | - |

## チャット関連コマンド

発言が届く範囲を切り替えるコマンドです。

| コマンド | タイトル | 説明 | 簡略化 |
|---|---|---|---|
| `cw:worldchat` | ワールドチャット | チャットの送信範囲をワールド全体（全員）にします。 | `cw:wchat` |
| `cw:countrychat` | 国チャット | チャットの送信範囲を同じ国のメンバーだけにします。 | `cw:cchat` |
| `cw:localchat` | ローカルチャット | チャットの送信範囲を近くのプレイヤーだけにします。 | `cw:lchat` |

## 管理者用コマンド

| コマンド | タイトル | 説明 | 簡略化 |
|---|---|---|---|
| `cw:menu` | 管理者メニュー | 管理者専用のツール画面を開きます。 | - |

::: tip 💡 簡略化コマンドとは？
一部のコマンドには「簡略化（短いコマンド）」が用意されています。
例えば `/makecountry` と入力する代わりに `/mc` と入力するだけで、同じように機能を使えるのでとても便利です！
:::
